1. Absolute Maxima and Minima

In business, we often don't just want a local peak or trough — we want the global best. For example, a company doesn't want to know that profit is higher on Tuesday than Monday; it wants to know when, over the entire year, profit is the highest. That is the idea of an absolute extremum.

Definition

Let \(f\) be a function defined on an interval \(I\).

• Absolute Maximum: \(f(c)\) is the absolute maximum of \(f\) on \(I\) if \(f(c) \geq f(x)\) for all \(x\) in \(I\).

• Absolute Minimum: \(f(c)\) is the absolute minimum of \(f\) on \(I\) if \(f(c) \leq f(x)\) for all \(x\) in \(I\).

Together, absolute maxima and minima are called absolute extrema.

Key Insight
Every relative maximum or minimum is a candidate for an absolute extremum, but the absolute extremum could also occur at the endpoints of the interval. Endpoints are sneaky — never forget to check them!

The Extreme Value Property

Theorem — Extreme Value Property

If \(f\) is continuous on a closed interval \([a,\, b]\), then \(f\) is guaranteed to attain both an absolute maximum and an absolute minimum on \([a,\, b]\).

This is a powerful guarantee: as long as your function has no holes or jumps on a closed, bounded interval, the best and worst values must exist somewhere on that interval.

How to Find Absolute Extrema on \([a,b]\)

Procedure
  1. Step 1: Find all critical numbers of \(f\) in the open interval \((a,b)\) — i.e., where \(f'(x) = 0\) or \(f'(x)\) is undefined.
  2. Step 2: Evaluate \(f(x)\) at each critical number found, and also at the endpoints \(x = a\) and \(x = b\).
  3. Step 3: The largest value from Step 2 is the absolute maximum; the smallest value is the absolute minimum.
Example 1 Maximizing Profit Over a Production Run

A factory's daily profit (in thousands of dollars) when producing \(x\) hundred units is modeled by:

$$P(x) = -x^3 + 6x^2 + 4, \quad 0 \leq x \leq 5$$

Find the absolute maximum and minimum profit over this production range.

Maximum profit is $40,000 at x = 4 (400 units); minimum is $4,000 at x = 0.
Step 1 — Find Critical Numbers

\(P'(x) = -3x^2 + 12x = -3x(x - 4)\)

Setting \(P'(x) = 0\): \(x = 0\) or \(x = 4\). Since \(x=0\) is an endpoint, the only interior critical number is \(x = 4\).

Step 2 — Evaluate at Critical Numbers and Endpoints

\(P(0) = 0 + 0 + 4 = 4\) (thousands)

\(P(4) = -64 + 96 + 4 = 36\) (thousands)

\(P(5) = -125 + 150 + 4 = 29\) (thousands)

Step 3 — Conclusion

The absolute maximum profit is \(\$36{,}000\) achieved when producing \(x = 4\) hundred units (400 units).

The absolute minimum profit is \(\$4{,}000\) at \(x = 0\) (no production — only fixed costs).

Example 2 Minimizing Inventory Cost

A retailer's total inventory cost (in hundreds of dollars) is given by:

$$C(x) = x^2 - 8x + 20, \quad 1 \leq x \leq 6$$

where \(x\) is the number of orders placed per month. Find when cost is minimized and maximized.

Minimum cost is $400 at x = 4; maximum cost is $1,300 at x = 1.
Step 1 — Find Critical Numbers

\(C'(x) = 2x - 8 = 0 \implies x = 4\)

\(x = 4\) is in \((1,6)\), so it is a valid critical number.

Step 2 — Evaluate

\(C(1) = 1 - 8 + 20 = 13\) (hundreds) = $1,300

\(C(4) = 16 - 32 + 20 = 4\) (hundreds) = $400

\(C(6) = 36 - 48 + 20 = 8\) (hundreds) = $800

Conclusion

Minimum cost of $400 is at \(x = 4\) orders/month. Maximum cost of $1,300 occurs at \(x = 1\) order/month (too infrequent — storage costs accumulate).

2. More General Optimization

The closed-interval method works beautifully when the domain is \([a,b]\). But in real business problems, the domain is often unbounded — for example, \(x > 0\) (you can't have negative production) with no upper ceiling specified. In such cases:

  • The Extreme Value Property no longer guarantees an extremum exists.
  • But if an absolute extremum exists and the function is continuous, it must occur at a relative extremum or an endpoint on the interval.
  • You must use judgment (the behavior as \(x \to \infty\)) to confirm what you found is truly the absolute extremum.

The Second Derivative Test for Absolute Extrema

Theorem

Suppose \(f(x)\) is continuous on an interval \(I\) where \(x = c\) is the only critical number, and \(f'(c) = 0\). Then:

• If \(f''(c) > 0\), then \(f(c)\) is the absolute minimum of \(f\) on \(I\).

• If \(f''(c) < 0\), then \(f(c)\) is the absolute maximum of \(f\) on \(I\).

Why this works
If there is only one critical point and the function curves upward there (\(f'' > 0\)), then the function was decreasing before and increasing after — this local minimum has no competition and must be the global minimum.
Example 3 Minimizing Average Cost of Production

A manufacturer's total cost (in dollars) is:

$$C(q) = \frac{q^5}{2} - 2q^4 - \frac{27}{2}q^3 + 81q^2 + 31q, \quad q > 0$$

Find the level of output \(q\) that minimizes the average cost per unit, defined as \(A(q) = C(q)/q\).

Minimum average cost ≈ $62.2 per unit occurs at q ≈ 2.19 units.
Set up A(q)

Given \(C(q) = \frac{q^5}{2} - 2q^4 - \frac{27}{2}q^3 + 81q^2 + 31q\),

$$ A(q) = \frac{C(q)}{q} = \frac{1}{2}q^4 - 2q^3 - \frac{27}{2}q^2 + 81q + 31 $$

Find Critical Numbers

Differentiate:

$$ A'(q) = 2q^3 - 6q^2 - 27q + 81 $$

Set \(A'(q) = 0\):

$$ 2q^3 - 6q^2 - 27q + 81 = 0 $$

Factor by grouping:

$$ 2q^2(q - 3) - 27(q - 3)=0 $$

$$ (q - 3)(2q^2 - 27) = 0 $$

So:

  • \(q = 3\)
  • \(2q^2 - 27 = 0 \Rightarrow q = \pm \sqrt{\frac{27}{2}} \approx \pm 3.67\)

Since \(q > 0\), the relevant critical points are:

\(q \approx 3\) and \(q \approx 3.67\)

Classify Using Second Derivative

$$ A''(q) = 6q^2 - 12q - 27 $$

Evaluate:

  • \(A''(3) = 54 - 36 - 27 = -9 < 0\) ⇒ local maximum
  • \(A''(3.67) > 0\) ⇒ local minimum

So the minimum occurs at \(q \approx 3.67\).

Compute Minimum Average Cost

$$ A(3.67) \approx \frac{1}{2}(3.67)^4 - 2(3.67)^3 - \frac{27}{2}(3.67)^2 + 81(3.67) + 31 $$

$$ \approx 90.6 - 98.6 - 181.7 + 297.3 + 31 \approx 138.6 $$

Minimum average cost is approximately $138.6 per unit at \(q \approx 3.67\).

Example 4 Optimal Pricing for a Coffee Shop

A coffee shop finds that when it charges \(p\) dollars per cup, it sells \(q = 200 - 40p\) cups per day. Each cup costs $1 to make, and daily fixed overhead is $60. Find the price that maximizes daily profit.

Maximum profit of $100 per day is achieved by charging $3 per cup.
Set up Revenue and Cost

Revenue: \(R = p \cdot q = p(200 - 40p) = 200p - 40p^2\)

Variable cost: \(\$1 \times q = 200 - 40p\)

Total cost: \(C = (200 - 40p) + 60 = 260 - 40p\)

Write Profit Function

\(\pi(p) = R - C = (200p - 40p^2) - (260 - 40p)\)

\(\pi(p) = -40p^2 + 240p - 260\)

Optimize

\(\pi'(p) = -80p + 240 = 0 \implies p = 3\)

\(\pi''(p) = -80 < 0\) → confirms this is a maximum.

Compute Maximum Profit

\(\pi(3) = -40(9) + 240(3) - 260 = -360 + 720 - 260 = 100\)

Maximum daily profit: $100 at price \(p = \$3\) per cup, selling \(200 - 120 = 80\) cups/day.

3. Optimization Principles in Economics

Economists express optimal business decisions in terms of marginal quantities — rates of change. The two key principles concern maximum profit and minimum average cost.

3a. Marginal Analysis Criterion for Maximum Profit

Principle

Profit \(P(q) = R(q) - C(q)\) is maximized at the output level \(q\) where:

Marginal Revenue = Marginal Cost   i.e.,   \(R'(q) = C'(q)\)

provided that \(R''(q) < C''(q)\) (so it's indeed a maximum, not a minimum).

Intuition: If producing one more unit brings in more revenue than it costs (\(MR > MC\)), you should produce more. If it costs more than it earns (\(MR < MC\)), you should produce less. Equilibrium is when they're equal.

Example 5 Maximizing Profit Using MR = MC

A tech startup sells software licenses. The revenue and cost functions (in thousands of dollars) are:

$$R(q) = 14q - \frac{q^2}{2} \qquad C(q) = \frac{q^3}{3} - 3q^2 + 5q + 10$$

where \(q\) is in hundreds of licenses. Find the profit-maximizing output.

Maximum profit occurs at q = 3 (300 licenses).
Compute Marginals

\(MR = R'(q) = 14 - q\)

\(MC = C'(q) = q^2 - 6q + 5\)

Set MR = MC

\(14 - q = q^2 - 6q + 5\)

\(q^2 - 5q - 9 = 0\)

Using the quadratic formula: \(q = \dfrac{5 \pm \sqrt{25 + 36}}{2} = \dfrac{5 \pm \sqrt{61}}{2}\)

Positive root: \(q \approx 6.4\). The key method: solve \(MR = MC\) algebraically and verify with \(P''(q) < 0\).

Verify Maximum

\(P''(q) = R''(q) - C''(q) = -1 - (2q - 6)\)

At the critical point, verify \(P''(q) < 0\) to confirm it's a maximum.

Example 6 Bakery Decision: How Many Loaves to Bake?

A bakery can sell loaves at a price determined by the demand function \(p = 8 - 0.5q\). Total cost is \(C(q) = q^2 + 2q + 1\). How many loaves \(q\) maximizes profit?

Optimal quantity is q = 2 loaves (batches), giving maximum profit of $5.
Revenue Function

\(R(q) = pq = (8 - 0.5q)q = 8q - 0.5q^2\)

Marginals

\(MR = R'(q) = 8 - q\)

\(MC = C'(q) = 2q + 2\)

MR = MC

\(8 - q = 2q + 2 \implies 6 = 3q \implies q = 2\)

Check & Compute Profit

\(P''(q) = R'' - C'' = -1 - 2 = -3 < 0\) ✓ (maximum)

\(R(2) = 16 - 2 = 14\)

\(C(2) = 4 + 4 + 1 = 9\)

\(P(2) = 14 - 9 = 5\)

Maximum profit is $5 at \(q = 2\).

3b. Marginal Analysis Criterion for Minimum Average Cost

Principle

The average cost \(A(q) = C(q)/q\) is minimized when:

Marginal Cost = Average Cost   i.e.,   \(C'(q) = \dfrac{C(q)}{q}\)

Intuition: When the cost of producing one more unit equals the average cost so far, you're at the "sweet spot" of efficiency. If MC is below the average, producing more lowers the average; if MC is above, it drags the average up.

Example 7 Minimum Average Cost for a Furniture Maker

A furniture company's total cost (in hundreds of dollars) for producing \(q\) chairs is:

$$C(q) = 2q^3 - 9q^2 + 12q + 8, \quad q > 0$$

Find the output that minimizes average cost per chair.

Minimum average cost occurs near q = 2 chairs (batches).
Average Cost Function

\(A(q) = \dfrac{C(q)}{q} = 2q^2 - 9q + 12 + \dfrac{8}{q}\)

Find Critical Point

\(A'(q) = 4q - 9 - \dfrac{8}{q^2}\)

Multiplying by \(q^2\): \(4q^3 - 9q^2 - 8 = 0\)

Alternative — Use MC = AC

\(MC = C'(q) = 6q^2 - 18q + 12\)

Set \(MC = AC\): \(6q^2 - 18q + 12 = 2q^2 - 9q + 12 + \dfrac{8}{q}\)

\(4q^2 - 9q = \dfrac{8}{q} \implies 4q^3 - 9q^2 - 8 = 0\)

Verify minimum by checking \(A''(q) > 0\) at the solution.

Economic Interpretation

Once minimum average cost is found, if selling above this level, the firm makes a profit; if below, it should reconsider production scale.

4. Price Elasticity of Demand

When a coffee shop raises prices by 10%, how much does demand drop? When a pharmacy raises insulin prices by 10%, demand barely changes. This sensitivity of quantity demanded to price changes is captured by the price elasticity of demand.

Definition — Price Elasticity of Demand

If \(q = D(p)\) units are demanded at unit price \(p\), the price elasticity of demand is:

$$E(p) = -\frac{p}{q} \cdot \frac{dq}{dp}$$
The negative sign ensures E(p) is positive, since demand curves typically slope downward (dq/dp < 0)

Interpretation: \(E(p) \approx\) percentage rate of decrease in demand produced by a 1% increase in price.

Levels of Elasticity

Condition Name Meaning Real-World Example
\(E(p) > 1\) Elastic % drop in demand > % rise in price. Consumers are sensitive. Luxury goods, airline tickets, brand-name electronics
\(E(p) < 1\) Inelastic % drop in demand < % rise in price. Consumers are insensitive. Insulin, gasoline, salt, basic utilities
\(E(p) = 1\) Unit Elastic % drop in demand = % rise in price. Revenue is at its peak here

Elasticity and Revenue

Key Principle — Levels of Elasticity and Revenue

• Elastic demand (\(E(p) > 1\)): Revenue \(R\) decreases as price \(p\) increases.

• Inelastic demand (\(E(p) < 1\)): Revenue \(R\) increases as price \(p\) increases.

• Unit elasticity (\(E(p) = 1\)): Revenue is unaffected by a small change in price — this is where revenue is maximized.

Practical Implication

If your product is inelastic (like a necessity), you can raise prices and still earn more revenue — customers will keep buying. If it is elastic (like a luxury), raising prices will hurt your revenue because customers switch to alternatives. Revenue is maximized at unit elasticity.

Example 8 Elasticity for a Linear Demand Curve

A grocery store finds that weekly demand for cooking oil is:

$$q = D(p) = 300 - 20p$$

(a) Find the elasticity \(E(p)\).   (b) Is demand elastic or inelastic at \(p = 5\)?   (c) At what price is revenue maximized?

E(5) = 1/2 (inelastic); revenue is maximized at p = $7.50 (unit elasticity).
Part (a) — Find E(p)

\(\dfrac{dq}{dp} = -20\)

\(E(p) = -\dfrac{p}{q}\cdot\dfrac{dq}{dp} = -\dfrac{p}{300 - 20p}\cdot(-20) = \dfrac{20p}{300 - 20p}\)

Simplified: \(E(p) = \dfrac{p}{15 - p}\)

Part (b) — Elasticity at p = 5

\(E(5) = \dfrac{5}{15 - 5} = \dfrac{5}{10} = 0.5\)

Since \(E(5) = 0.5 < 1\), demand is inelastic at $5. Raising price will increase revenue.

Part (c) — Revenue Maximization

Revenue is maximized when \(E(p) = 1\):

\(\dfrac{p}{15 - p} = 1 \implies p = 15 - p \implies 2p = 15 \implies p = 7.50\)

At this price: \(q = 300 - 150 = 150\) units.

\(R = 7.50 \times 150 = \$1{,}125\) per week (maximum revenue).

Example 9 Elasticity for a Nonlinear Demand

An online retailer's demand function for a tablet case is:

$$q = D(p) = \frac{500}{p^2}$$

(a) Find \(E(p)\).   (b) Is demand elastic, inelastic, or unit elastic?   (c) What happens to revenue if you raise price?

E(p) = 2 (elastic) for all p. Raising price always decreases revenue.
Compute the derivative

\(q = 500p^{-2} \implies \dfrac{dq}{dp} = -1000p^{-3} = -\dfrac{1000}{p^3}\)

Compute E(p)

\(E(p) = -\dfrac{p}{q}\cdot\dfrac{dq}{dp} = -\dfrac{p}{\,500/p^2\,}\cdot\left(-\dfrac{1000}{p^3}\right) = -\dfrac{p^3}{500}\cdot\left(-\dfrac{1000}{p^3}\right) = \dfrac{1000}{500} = 2\)

Interpretation

\(E(p) = 2 > 1\) → demand is elastic for all prices.

A 1% increase in price causes a 2% decrease in demand. Since demand is elastic, raising price reduces revenue. The retailer should lower prices to increase revenue.

Example 10 A Pharmacy's Pricing Dilemma

A pharmacy sells a common antibiotic at price \(p\) dollars. Market research shows demand is \(q = 800 - 50p\) units per month. The pharmacist is considering raising the price from $8 to $10.

(a) Calculate \(E(8)\) and \(E(10)\).

(b) Should the pharmacist raise the price to increase revenue? Explain using elasticity.

E(8) = 1 (unit elastic); E(10) ≈ 1.67 (elastic). Revenue is already maximum at $8 — raising price decreases revenue.
Elasticity Formula

\(\dfrac{dq}{dp} = -50\)

\(E(p) = -\dfrac{p}{q}\cdot\dfrac{dq}{dp} = \dfrac{50p}{800 - 50p}\)

Part (a)

\(E(8) = \dfrac{400}{800 - 400} = \dfrac{400}{400} = 1\)

\(E(10) = \dfrac{500}{800 - 500} = \dfrac{500}{300} \approx 1.67\)

Part (b) — Decision

At \(p = \$8\): \(E = 1\) → unit elastic → revenue is already at maximum!

\(R(8) = 8 \times 400 = \$3{,}200\)

At \(p = \$10\): \(E \approx 1.67 > 1\) → elastic → revenue falls.

\(R(10) = 10 \times 300 = \$3{,}000\)

Conclusion: The pharmacist should not raise the price. At $8, revenue is already at its maximum of $3,200. Raising to $10 reduces revenue by $200/month because demand becomes elastic beyond that point.

Exercise 1 — Absolute Extrema on a Closed Interval

Find the absolute maximum and minimum values of \(f(x) = x^3 - 3x^2 - 9x + 5\) on \([-2,\, 6]\).

Maximum is 59 at x = 6; minimum is −22 at x = 3.
Step 1 — Critical Numbers

\(f'(x) = 3x^2 - 6x - 9 = 3(x^2 - 2x - 3) = 3(x-3)(x+1)\)

Critical numbers: \(x = 3\) and \(x = -1\) (both in \([-2, 6]\)).

Step 2 — Evaluate

\(f(-2) = -8 - 12 + 18 + 5 = 3\)

\(f(-1) = -1 - 3 + 9 + 5 = 10\)

\(f(3) = 27 - 27 - 27 + 5 = -22\)

\(f(6) = 216 - 108 - 54 + 5 = 59\)

Conclusion

Absolute Maximum: 59 at \(x = 6\) (endpoint!)

Absolute Minimum: −22 at \(x = 3\)

Note how the maximum occurred at an endpoint — always check them!

Exercise 2 — Business Optimization

A company's weekly profit (in thousands of dollars) from selling \(x\) units is: \(P(x) = -2x^3 + 12x^2 - 18x + 5\) for \(0 \leq x \leq 5\). Find the production level that maximizes profit.

Maximum profit is $5,000 at x = 3 (or x = 0).
Find Critical Numbers

\(P'(x) = -6x^2 + 24x - 18 = -6(x^2 - 4x + 3) = -6(x-1)(x-3)\)

Critical numbers: \(x = 1\) and \(x = 3\)

Evaluate at All Candidates

\(P(0) = 5\)

\(P(1) = -2 + 12 - 18 + 5 = -3\)

\(P(3) = -54 + 108 - 54 + 5 = 5\)

\(P(5) = -250 + 300 - 90 + 5 = -35\)

Conclusion

Maximum profit of $5,000 is achieved at both \(x = 0\) and \(x = 3\). The practical answer is \(x = 3\). Minimum is at \(x = 5\): a loss of $35,000 — don't overproduce!

Exercise 3 — Elasticity

The demand for concert tickets is \(q = D(p) = 1200 - 60p\).

(a) Compute \(E(p)\).   (b) At what price is demand unit elastic?   (c) At \(p = 5\), is it better to raise or lower the price to increase revenue?

E(p) = p/(20−p); unit elastic at p = $10; at p = $5, demand is inelastic — raise price to increase revenue.
Part (a)

\(\dfrac{dq}{dp} = -60\)

\(E(p) = -\dfrac{p}{q}\cdot\dfrac{dq}{dp} = \dfrac{60p}{1200 - 60p} = \dfrac{p}{20 - p}\)

Part (b) — Unit Elasticity

\(E(p) = 1 \implies \dfrac{p}{20-p} = 1 \implies p = 20 - p \implies p = \$10\)

Revenue is maximized at \(p = \$10\).

Part (c) — At p = 5

\(E(5) = \dfrac{5}{15} = \dfrac{1}{3} < 1\) → demand is inelastic

Since demand is inelastic at $5, the concert venue should raise the price to increase total revenue.

Exercise 4 — Marginal Analysis: MR = MC

A firm sells widgets with demand \(p = 50 - 2q\) and total cost \(C(q) = q^2 + 10q + 100\). Find the profit-maximizing quantity using MR = MC.

Profit is maximized at q ≈ 6.67 units.
Revenue and Marginals

\(R(q) = pq = (50 - 2q)q = 50q - 2q^2\)

\(MR = R'(q) = 50 - 4q\)

\(MC = C'(q) = 2q + 10\)

Set MR = MC

\(50 - 4q = 2q + 10 \implies 40 = 6q \implies q = \dfrac{40}{6} \approx 6.67\)

Verify Maximum

\(P''(q) = MR' - MC' = -4 - 2 = -6 < 0\) ✓

Compute Profit

At \(q \approx 6.67\): \(p = 50 - 13.33 = 36.67\)

\(R \approx 36.67 \times 6.67 \approx \$244.5\)

\(C \approx 44.5 + 66.7 + 100 = \$211.2\)

Profit \(\approx \$33.3\)

Exercise 5 — Scenario: Smartphone App Pricing

A mobile app company sells subscriptions at price \(p\) dollars/month. Their demand function is \(q = 10{,}000 \cdot p^{-1.5}\).

(a) Find \(E(p)\).   (b) At \(p = \$9\), is demand elastic or inelastic?   (c) Should they raise or lower price to maximize revenue?

E(p) = 1.5 (elastic for all p). Demand is always elastic — lowering price increases revenue.
Part (a) — Compute E(p)

\(q = 10000 \cdot p^{-1.5}\)

\(\dfrac{dq}{dp} = 10000 \cdot (-1.5) \cdot p^{-2.5} = -15000 \cdot p^{-2.5}\)

\(E(p) = -\dfrac{p}{q}\cdot\dfrac{dq}{dp} = -\dfrac{p}{10000p^{-1.5}}\cdot(-15000p^{-2.5}) = \dfrac{15000\, p^{2.5}}{10000\, p^{2.5}} = 1.5\)

Part (b) & (c)

\(E(p) = 1.5 > 1\) for all prices → demand is always elastic.

Since demand is elastic, raising price always decreases revenue. The company should consider lowering price to grow revenue through volume.

Note: for a power demand \(q = Ap^{-n}\), the elasticity is always the constant \(n\).

Exercise 6 — Challenge: Combining Optimization and Elasticity

A hotel chain's nightly room demand is \(q = 400 - 10p\) and total operating cost is \(C(q) = 5q + 200\).

(a) Find the revenue-maximizing price using elasticity.   (b) Find the profit-maximizing price using \(MR = MC\).   (c) Are these the same? Why or why not?

Revenue max at p = $20; profit max at p = $22.50. They differ because profit accounts for costs.
Part (a) — Revenue Maximizing Price

\(\dfrac{dq}{dp} = -10\)

\(E(p) = \dfrac{10p}{400 - 10p} = \dfrac{p}{40 - p}\)

Set \(E(p) = 1\): \(p = 40 - p \implies p = \$20\)

Revenue-maximizing: \(p = \$20\), \(q = 200\) rooms, \(R = \$4{,}000\)/night.

Part (b) — Profit Maximizing Price

From \(q = 400 - 10p \implies p = 40 - \dfrac{q}{10}\)

\(R(q) = \left(40 - \dfrac{q}{10}\right)q = 40q - \dfrac{q^2}{10}\)

\(MR = 40 - \dfrac{q}{5}, \quad MC = C'(q) = 5\)

Set \(MR = MC\): \(40 - \dfrac{q}{5} = 5 \implies q = 175\)

\(p = 40 - \dfrac{175}{10} = \$22.50\)

Part (c) — Comparison

The revenue-max price ($20) differs from the profit-max price ($22.50). Maximizing revenue ignores costs. Since each room costs $5 to service, the hotel needs to charge slightly more. A business should usually optimize profit, not just revenue.

Profit at \(q = 175\): \(R = 22.50 \times 175 = \$3{,}937.50\), \(C = 5(175)+200 = \$1{,}075\), \(P = \$2{,}862.50\).

Section Summary

Absolute Extrema: On a closed interval \([a,b]\), evaluate \(f\) at all critical numbers plus endpoints. The largest is the absolute max, smallest is the absolute min.

Second Derivative Test (open intervals): If \(f\) has one critical number \(c\) and \(f''(c) > 0\), that's an absolute min; if \(f''(c) < 0\), it's an absolute max.

Max Profit: Occurs where \(MR = MC\) (with \(R'' < C''\)).

Min Average Cost: Occurs where \(MC = AC\).

Elasticity: \(E(p) = -\dfrac{p}{q}\dfrac{dq}{dp}\). Elastic: \(E > 1\) (raise price → less revenue). Inelastic: \(E < 1\) (raise price → more revenue). Unit elastic: \(E = 1\) (revenue is maximized).

Problem 43 Hoffmann §3.4 · Exercise 43 Elasticity — Constant Elasticity Demand

Suppose the demand equation for a certain commodity is:

$$q = \frac{a}{p^m}$$

where \(a\) and \(m\) are positive constants. Show that the elasticity of demand equals \(m\) for all values of \(p\). Interpret this result.

E(p) = m (constant). This is a power demand function — its elasticity is always equal to the exponent m, regardless of price.
Rewrite q

\(q = a \cdot p^{-m}\)

Differentiate

\(\dfrac{dq}{dp} = a \cdot (-m) \cdot p^{-m-1} = -ma \cdot p^{-m-1}\)

Apply Elasticity Formula

\(E(p) = -\dfrac{p}{q}\cdot\dfrac{dq}{dp}\)

\(= -\dfrac{p}{a\,p^{-m}} \cdot \left(-ma\,p^{-m-1}\right)\)

\(= -\dfrac{p^{m+1}}{a} \cdot \left(-\dfrac{ma}{p^{m+1}}\right) = m \quad\checkmark\)

Interpretation

For a power-law demand function \(q = a/p^m\), the elasticity is the constant \(m\) — it doesn't change with price. This means:

  • If \(m > 1\): demand is always elastic
  • If \(m < 1\): demand is always inelastic
  • If \(m = 1\): demand is always unit elastic — revenue is constant for all prices!

$$Q(x) = \frac{R(x)}{C(x)}$$

is optimized (critical point occurs) when the relative rate of change of revenue equals the relative rate of change of cost. Would you expect this optimum to be a maximum or minimum?

Q'(x) = 0 leads to R'(x)/R(x) = C'(x)/C(x). This is typically a maximum (maximizing return-on-cost ratio).
Differentiate Q(x) using the Quotient Rule

\(Q'(x) = \dfrac{R'(x)\cdot C(x) - R(x)\cdot C'(x)}{[C(x)]^2}\)

Set Q'(x) = 0

The denominator \([C(x)]^2 > 0\) always, so:

\(R'(x)\cdot C(x) - R(x)\cdot C'(x) = 0\)

\(R'(x)\cdot C(x) = R(x)\cdot C'(x)\)

Rearrange

Divide both sides by \(R(x)\cdot C(x) > 0\):

\(\dfrac{R'(x)}{R(x)} = \dfrac{C'(x)}{C(x)}\)

This is exactly the condition that the relative rate of change of revenue equals the relative rate of change of cost. ✓

Maximum or Minimum?

\(Q(x) = R(x)/C(x)\) measures revenue earned per dollar of cost — a profitability ratio. As \(x \to 0\) or \(x \to \infty\), \(Q \to 0\), so the interior critical point is a maximum.

Problem 46 Hoffmann §3.4 · Exercise 46 Growth of a Species — Moths Surviving Pupation

The percentage of coding moths that survive the pupa stage at temperature \(T\) (°C) is:

Maximum survival (~68.4%) at T ≈ 23.9°C; minimum survival at T = 30°C (~16%).

Problem 47 Hoffmann §3.4 · Exercise 47 Blood Circulation — Maximum Speed in an Artery

Poiseuille's law asserts that the speed of blood \(r\) centimeters from the centre of an artery of radius \(R\) is:

$$S(r) = c(R^2 - r^2)$$

where \(c\) is a positive constant. Where is the speed of blood the greatest?

Blood speed is greatest at r = 0 (the centre of the artery).
Domain

The variable \(r\) represents distance from the centre, so \(0 \leq r \leq R\).

Find Critical Numbers

\(S'(r) = c \cdot (-2r) = -2cr\)

Set \(S'(r) = 0\): \(-2cr = 0 \implies r = 0\) (the left endpoint).

Evaluate at Endpoints

\(S(0) = c(R^2 - 0) = cR^2\)

\(S(R) = c(R^2 - R^2) = 0\)

Conclusion

Maximum blood speed is \(cR^2\), achieved at \(r = 0\), the center of the artery. Blood velocity is zero at the artery walls (\(r = R\)) due to friction. This is laminar (Poiseuille) flow.

Problem 49 Hoffmann §3.4 · Exercise 49 Learning — Maximum Likelihood Estimate

In a learning model, two responses (A and B) are possible. The probability of getting response A exactly \(n\) times in a series of \(m\) observations is:

$$F(p) = p^n(1-p)^{m-n}, \quad 0 \leq p \leq 1$$

The maximum likelihood estimate is the value of \(p\) that maximizes \(F(p)\). Find this value.

p = n/m (the observed proportion of A responses — the sample frequency is the maximum likelihood estimator).
Differentiate F(p) using Product Rule

\(F(p) = p^n \cdot (1-p)^{m-n}\)

\(F'(p) = n\,p^{n-1}(1-p)^{m-n} + p^n(m-n)(1-p)^{m-n-1}(-1)\)

Factor F'(p)

\(F'(p) = p^{n-1}(1-p)^{m-n-1}\bigl[n(1-p) - p(m-n)\bigr]\)

Set F'(p) = 0

Since \(p^{n-1}(1-p)^{m-n-1} > 0\) for \(0 < p < 1\), we need:

\(n(1-p) - p(m-n) = 0\)

\(n - np - pm + pn = 0 \implies n - pm = 0 \implies p = \dfrac{n}{m}\) ✓

Interpretation

The maximum likelihood estimate is \(p = n/m\) — simply the proportion of times response A was observed. Since \(F(0) = F(1) = 0\) and \(F(p) > 0\) on \((0,1)\), this critical point must be a maximum.

Problem 50 Hoffmann §3.4 · Exercise 50 Voting Pattern — House Seat Proportion (Cube Rule)

After a presidential election, the proportion \(h(p)\) of seats in the House of Representatives won by the winning candidate's party is modeled by the cube rule:

$$h(p) = \frac{p^3}{p^3 + (1-p)^3}, \quad 0 \leq p \leq 1$$

where \(p\) is the proportion of the popular vote received.

(a) Find \(h'(p)\) and \(h''(p)\).   (b) Sketch the graph of \(h(p)\).   (c) In 1964, Lyndon Johnson received 61% of the popular vote. What percentage of House seats does the cube rule predict?

(c) h(0.61) ≈ 79.3% — remarkably close to the actual result of 72%!
Part (a) — Compute \(h'(p)\) and \(h''(p)\)

Given \(h(p) = \dfrac{p^3}{p^3 + (1-p)^3}\), let:

\(u = p^3,\quad v = p^3 + (1-p)^3\)

Step 1: First Derivative \(h'(p)\)

Using the quotient rule:

$$h'(p) = \frac{u'v - uv'}{v^2}$$

Compute derivatives:

\(u' = 3p^2\)

\(v' = 3p^2 - 3(1-p)^2 = 3(2p - 1)\)

Substitute:

$$ h'(p) = \frac{3p^2\big(p^3 + (1-p)^3\big) - p^3 \cdot 3(2p-1)}{\big[p^3 + (1-p)^3\big]^2} $$

Simplifying:

$$ h'(p) = \frac{3p^2(1-p)^2}{\big[p^3 + (1-p)^3\big]^2} $$

Step 2: Second Derivative \(h''(p)\)

Differentiate \(h'(p)\). After simplification:

$$ h''(p) = \frac{6p(1-p)(1-2p)}{\big[p^3 + (1-p)^3\big]^3} $$

Conclusion:

  • \(h'(p) > 0\) for \(0 < p < 1 \implies \) function is increasing
  • \(h''(p) = 0\) at \(p = 0.5 \implies \) inflection point
Part (b) — Graph Sketch Using Calculus Rules

Domain: \(0 \leq p \leq 1\)

Intercepts: \((0,0)\), \((1,1)\)

Monotonicity: Increasing on entire domain

Concavity: Concave up on \((0,0.5)\), concave down on \((0.5,1)\)

Inflection Point: \((0.5,0.5)\)

Asymptotes: None

Final Graph:

Part (c) — 1964 Election: p = 0.61

\(p^3 = (0.61)^3 = 0.226981\)

\((1-p)^3 = (0.39)^3 = 0.059319\)

\(h(0.61) = \dfrac{0.226981}{0.226981 + 0.059319} \approx 79.3\%\)

They actually won 72%. The cube rule captures the amplification effect well.

Problem 51 Hoffmann §3.4 · Exercise 51 Ornithology — Energy Expenditure of a Parakeet

According to Tucker and Schmidt-Koenig, the energy expended by a certain species of parakeet is:

$$E(v) = \frac{1}{v}\left[0.074(v-35)^2 + 22\right]$$

where \(v\) is the bird's speed (km/h). Find the speed that minimizes energy expenditure.

Optimal cruising speed is approximately v ≈ 39 km/h.
Expand E(v)

\(E(v) = \dfrac{0.074(v-35)^2 + 22}{v}\)

Expand \((v-35)^2 = v^2 - 70v + 1225\):

\(E(v) = 0.074\!\left(v - 70 + \dfrac{1225}{v}\right) + \dfrac{22}{v} = 0.074v - 5.18 + \dfrac{90.65}{v} + \dfrac{22}{v}\)

\(E(v) = 0.074v - 5.18 + \dfrac{112.65}{v}\)

Differentiate and Set E'(v) = 0

\(E'(v) = 0.074 - \dfrac{112.65}{v^2} = 0\)

\(v^2 = \dfrac{112.65}{0.074} = 1522.3 \implies v = \sqrt{1522.3} \approx 39.0 \text{ km/h}\)

Confirm Minimum

\(E''(v) = \dfrac{225.3}{v^3} > 0\) for all \(v > 0\) → this is a minimum. ✓

Conclusion

The parakeet minimizes energy expenditure at approximately \(v \approx 39\) km/h. Flying slower or faster than this costs more energy per unit distance.

Problem 37 Hoffmann §3.4 · Exercise 37 Production Control

Gina manages a company that makes two toys: an inexpensive doll (Floppsy) at \(x\) hundred units, and an expensive doll (Moppsy) at \(y\) hundreds. The company can produce them such that:

$$y = \frac{82 - 10x}{10 - x}, \quad 0 \leq x \leq 8$$

The company receives twice as much from selling a Moppsy doll as a Floppsy doll. How many of each kind should Gina recommend to maximize total revenue?

Maximum revenue at x = 4 (400 Floppsy), y = 7 (700 Moppsy).
Set Up Revenue

Let price of Floppsy = $1 (relative unit). Then Moppsy = $2.

\(R(x) = x + 2y = x + \dfrac{2(82-10x)}{10-x} = \dfrac{-x^2 - 10x + 164}{10-x}\)

Differentiate Using Quotient Rule

Let \(N = -x^2 - 10x + 164\), \(D = 10-x\).

\(R'(x) = \dfrac{N'D - ND'}{D^2} = \dfrac{(-2x-10)(10-x) - (-x^2-10x+164)(-1)}{(10-x)^2}\)

After expanding: numerator \(= x^2 - 20x + 64\)

\(R'(x) = \dfrac{x^2 - 20x + 64}{(10-x)^2}\)

Solve R'(x) = 0

\(x^2 - 20x + 64 = 0 \implies x = \dfrac{20 \pm \sqrt{400 - 256}}{2} = \dfrac{20 \pm 12}{2}\)

\(x = 16\) (outside domain) or \(x = 4\)

Evaluate at x = 4 and Endpoints

\(y(4) = (82-40)/(10-4) = 42/6 = 7\); \(R(4) = 4 + 14 = 18\)

\(y(0) = 8.2\); \(R(0) = 0 + 16.4 = 16.4\)

\(y(8) = 1\); \(R(8) = 8 + 2 = 10\)

Maximum Revenue: 18 units at \(x = 4\) (400 Floppsy) and \(y = 7\) (700 Moppsy).

Problem 38 Hoffmann §3.4 · Exercise 38 National Consumption

Assume that total national consumption is given by a function \(C(x)\), where \(x\) is total national income. The derivative \(C'(x)\) is called the marginal propensity to consume. Total savings is:

$$S(x) = x - C(x)$$

and \(S'(x)\) is called the marginal propensity to save. Given:

$$C(x) = 8 + 0.8x + 0.8\sqrt{x}$$

Find the marginal propensity to consume and the value of \(x\) that minimizes total savings.

MPC: C'(x) = 0.8 + 0.4/√x; Minimum savings at x = 4.
Step 1: Marginal Propensity to Consume

\(C(x) = 8 + 0.8x + 0.8x^{1/2}\)

\(C'(x) = 0.8 + 0.8 \cdot \dfrac{1}{2} x^{-1/2} = 0.8 + \dfrac{0.4}{\sqrt{x}}\)

Step 2: Savings Function

\(S(x) = x - C(x) = 0.2x - 0.8\sqrt{x} - 8\)

Step 3: Minimize Savings

\(S'(x) = 0.2 - \dfrac{0.4}{\sqrt{x}} = 0 \implies \sqrt{x} = 2 \implies x = 4\)

Conclusion

Marginal propensity to consume: \(C'(x) = 0.8 + \dfrac{0.4}{\sqrt{x}}\)

Minimum savings occurs at \(\mathbf{x = 4}\).

Problem 39 Hoffmann §3.4 · Exercise 39 Worker Efficiency

An efficiency study shows that an average worker starting at 8:00 AM assembles:

$$f(x) = -x^3 + 6x^2 + 15x$$

After a 15-minute coffee break, the worker assembles:

$$g(x) = -\tfrac{1}{3}x^3 + x^2 + 23x$$

If the break begins \(t\) hours after 8:00 AM, total production is \(T(t) = f(t) + g(4-t)\). Find the optimal break time.

Optimal break time: t = 3 hours → 11:00 AM
Step 1: Expand g(4 − t)

\((4-t)^3 = 64 - 48t + 12t^2 - t^3\)

\(g(4-t) = -\tfrac{1}{3}(64-48t+12t^2-t^3) + (4-t)^2 + 23(4-t)\)

\(= \tfrac{1}{3}t^3 - 3t^2 - 15t + \tfrac{260}{3}\)

Step 2: Total Function

\(T(t) = f(t) + g(4-t) = (-t^3 + 6t^2 + 15t) + \left(\tfrac{1}{3}t^3 - 3t^2 - 15t + \tfrac{260}{3}\right)\)

\(T(t) = -\tfrac{2}{3}t^3 + 3t^2 + \tfrac{260}{3}\)

Step 3: Differentiate and Find Critical Points

\(T'(t) = -2t^2 + 6t = -2t(t-3) = 0 \implies t = 0 \text{ or } t = 3\)

Step 4: Confirm Maximum

\(T''(t) = -4t + 6\). At \(t = 3\): \(T''(3) = -6 < 0\) ✓ (maximum)

Optimal break time: 11:00 AM (3 hours after 8:00 AM).
