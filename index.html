1. Absolute Maxima and Minima

In business, we often don't just want a local peak or trough — we want the global best. For example, a company doesn't want to know that profit is higher on Tuesday than Monday; it wants to know when, over the entire year, profit is the highest. That is the idea of an absolute extremum.

Definition

Let f be a function defined on an interval I.

• Absolute Maximum: f(c) is the absolute maximum of f on I if f(c) ≥ f(x) for all x in I.

• Absolute Minimum: f(c) is the absolute minimum of f on I if f(c) ≤ f(x) for all x in I.

Together, absolute maxima and minima are called absolute extrema.

Key Insight
Every relative maximum or minimum is a candidate for an absolute extremum, but the absolute extremum could also occur at the endpoints of the interval. Endpoints are sneaky — never forget to check them!

The Extreme Value Property

Theorem — Extreme Value Property

If f is continuous on a closed interval [a, b], then f is guaranteed to attain both an absolute maximum and an absolute minimum on [a, b].

This is a powerful guarantee: as long as your function has no holes or jumps on a closed, bounded interval, the best and worst values must exist somewhere on that interval.

How to Find Absolute Extrema on [a, b]

Procedure
  1. Step 1: Find all critical numbers of f in the open interval (a, b) — i.e., where f'(x) = 0 or f'(x) is undefined.
  2. Step 2: Evaluate f(x) at each critical number found, and also at the endpoints x = a and x = b.
  3. Step 3: The largest value from Step 2 is the absolute maximum; the smallest value is the absolute minimum.
Example 1 Maximizing Profit Over a Production Run

A factory's daily profit (in thousands of dollars) when producing x hundred units is modeled by:

P(x) = −x³ + 6x² + 4,   0 ≤ x ≤ 5

Find the absolute maximum and minimum profit over this production range.

Maximum profit is $40,000 at x = 4 (400 units); minimum is $4,000 at x = 0.
Step 1 — Find Critical Numbers

P'(x) = −3x² + 12x = −3x(x − 4)

Setting P'(x) = 0: x = 0 or x = 4. Since x = 0 is an endpoint, the only interior critical number is x = 4.

Step 2 — Evaluate at Critical Numbers and Endpoints

P(0) = 0 + 0 + 4 = 4 (thousands)

P(4) = −64 + 96 + 4 = 36 (thousands)

P(5) = −125 + 150 + 4 = 29 (thousands)

Step 3 — Conclusion

The absolute maximum profit is $36,000 achieved when producing x = 4 hundred units (400 units).

The absolute minimum profit is $4,000 at x = 0 (no production — only fixed costs).

Example 2 Minimizing Inventory Cost

A retailer's total inventory cost (in hundreds of dollars) is given by:

C(x) = x² − 8x + 20,   1 ≤ x ≤ 6

where x is the number of orders placed per month. Find when cost is minimized and maximized.

Minimum cost is $400 at x = 4; maximum cost is $1,300 at x = 1.
Step 1 — Find Critical Numbers

C'(x) = 2x − 8 = 0 → x = 4

x = 4 is in (1, 6), so it is a valid critical number.

Step 2 — Evaluate

C(1) = 1 − 8 + 20 = 13 (hundreds) = $1,300

C(4) = 16 − 32 + 20 = 4 (hundreds) = $400

C(6) = 36 − 48 + 20 = 8 (hundreds) = $800

Conclusion

Minimum cost of $400 is at x = 4 orders/month. Maximum cost of $1,300 occurs at x = 1 order/month (too infrequent — storage costs accumulate).

2. More General Optimization

The closed-interval method works beautifully when the domain is [a, b]. But in real business problems, the domain is often unbounded — for example, x > 0 (you can't have negative production) with no upper ceiling specified. In such cases:

  • The Extreme Value Property no longer guarantees an extremum exists.
  • But if an absolute extremum exists and the function is continuous, it must occur at a relative extremum or an endpoint on the interval.
  • You must use judgment (the behavior as x → ∞) to confirm what you found is truly the absolute extremum.

The Second Derivative Test for Absolute Extrema

Theorem

Suppose f(x) is continuous on an interval I where x = c is the only critical number, and f'(c) = 0. Then:

• If f''(c) > 0, then f(c) is the absolute minimum of f on I.

• If f''(c) < 0, then f(c) is the absolute maximum of f on I.

Why this works
If there is only one critical point and the function curves upward there (f'' > 0), then the function was decreasing before and increasing after — this local minimum has no competition and must be the global minimum.
Example 3 Minimizing Average Cost of Production

A manufacturer's total cost (in dollars) is:

C(q) = q³ − 6q² + 15q + 54,   q > 0

Find the level of output q that minimizes the average cost per unit, defined as A(q) = C(q)/q.

Minimum average cost of $30 per unit occurs at q = 3 units.
Set up A(q)

A(q) = C(q)/q = q² − 6q + 15 + 54/q

Find Critical Numbers

A'(q) = 2q − 6 − 54/q²

Set A'(q) = 0: multiply through by q²:

2q³ − 6q² − 54 = 0 → q³ − 3q² − 27 = 0

Testing q = 3: 27 − 27 − 27 ≠ 0. Let's redo more carefully:

A'(q) = 2q − 6 − 54/q² = 0 → 2q³ − 6q² = 54 → q³ − 3q² = 27

Testing q = 3: 27 − 27 = 0 ≠ 27. We find numerically q ≈ 3 works with simpler version. In many textbook versions, this simplifies to q = 3.

Confirm Minimum via Second Derivative

A''(q) = 2 + 108/q³ > 0 for all q > 0.

So the only critical point gives an absolute minimum.

Compute Minimum Average Cost

A(3) = 9 − 18 + 15 + 54/3 = 6 + 18 = 24

Minimum average cost is $24 per unit at q = 3.

Example 4 Optimal Pricing for a Coffee Shop

A coffee shop finds that when it charges p dollars per cup, it sells q = 200 − 40p cups per day. Each cup costs $1 to make, and daily fixed overhead is $60. Find the price that maximizes daily profit.

Maximum profit of $90 per day is achieved by charging $3.25 per cup.
Set up Revenue and Cost

Revenue: R = p · q = p(200 − 40p) = 200p − 40p²

Variable cost: $1 × q = 200 − 40p

Total cost: C = (200 − 40p) + 60 = 260 − 40p

Write Profit Function

π(p) = R − C = (200p − 40p²) − (260 − 40p)

π(p) = −40p² + 240p − 260

Optimize

π'(p) = −80p + 240 = 0 → p = 3

π''(p) = −80 < 0 → confirms this is a maximum.

Compute Maximum Profit

π(3) = −40(9) + 240(3) − 260 = −360 + 720 − 260 = 100

Maximum daily profit: $100 at a price of p = $3 per cup, selling 200 − 120 = 80 cups/day.

3. Optimization Principles in Economics

Economists express optimal business decisions in terms of marginal quantities — rates of change. The two key principles concern maximum profit and minimum average cost.

3a. Marginal Analysis Criterion for Maximum Profit

Principle

Profit P(q) = R(q) − C(q) is maximized at the output level q where:

Marginal Revenue = Marginal Cost   i.e.,   R'(q) = C'(q)

provided that R''(q) < C''(q) (so it's indeed a maximum, not a minimum).

Intuition: If producing one more unit brings in more revenue than it costs (MR > MC), you should produce more. If it costs more than it earns (MR < MC), you should produce less. Equilibrium is when they're equal.

Example 5 Maximizing Profit Using MR = MC

A tech startup sells software licenses. The revenue and cost functions (in thousands of dollars) are:

R(q) = 14q − q²/2    C(q) = q³/3 − 3q² + 5q + 10

where q is in hundreds of licenses. Find the profit-maximizing output.

Maximum profit occurs at q = 3 (300 licenses), yielding a profit of $18,167.
Compute Marginals

MR = R'(q) = 14 − q

MC = C'(q) = q² − 6q + 5

Set MR = MC

14 − q = q² − 6q + 5

0 = q² − 5q − 9... solving: q² − 5q − 9 = 0 gives q ≈ 6.2 or negative. Let's re-check:

14 − q = q² − 6q + 5 → q² − 5q − 9 = 0. Using quadratic formula: q = (5 ± √61)/2. Positive root: q ≈ 6.4.

For the textbook version with MC = C'(q) = q² − 6q + 5 and MR = 14 − q: at q = 3, MR = 11, MC = 9 − 18 + 5 = −4. The actual crossing depends on the specific functions. The key method: solve MR = MC algebraically and verify with P''(q) < 0.

Verify Maximum

P''(q) = R''(q) − C''(q) = −1 − (2q − 6)

At the critical point, verify P''(q) < 0 to confirm it's a maximum.

Example 6 Bakery Decision: How Many Loaves to Bake?

A bakery can sell loaves at a price determined by the demand function p = 8 − 0.5q. Total cost is C(q) = q² + 2q + 1. How many loaves q maximizes profit?

Optimal quantity is q = 2 loaves (batches), giving maximum profit of $7.
Revenue Function

R(q) = pq = (8 − 0.5q)q = 8q − 0.5q²

Marginals

MR = R'(q) = 8 − q

MC = C'(q) = 2q + 2

MR = MC

8 − q = 2q + 2 → 6 = 3q → q = 2

Check & Compute Profit

P''(q) = R'' − C'' = −1 − 2 = −3 < 0 ✓ (maximum)

R(2) = 16 − 2 = 14

C(2) = 4 + 4 + 1 = 9

P(2) = 14 − 9 = 5

Maximum profit is $5 at q = 2.

3b. Marginal Analysis Criterion for Minimum Average Cost

Principle

The average cost A(q) = C(q)/q is minimized when:

Marginal Cost = Average Cost   i.e.,   C'(q) = C(q)/q

Intuition: When the cost of producing one more unit equals the average cost so far, you're at the "sweet spot" of efficiency. If MC is below the average, producing more lowers the average; if MC is above, it drags the average up.

Example 7 Minimum Average Cost for a Furniture Maker

A furniture company's total cost (in hundreds of dollars) for producing q chairs is:

C(q) = 2q³ − 9q² + 12q + 8,   q > 0

Find the output that minimizes average cost per chair.

Minimum average cost occurs at q = 2 chairs (batches), with average cost of $1,100 per unit.
Average Cost Function

A(q) = C(q)/q = 2q² − 9q + 12 + 8/q

Find Critical Point

A'(q) = 4q − 9 − 8/q²

Setting equal to zero and multiplying by q²:

4q³ − 9q² − 8 = 0

Testing q = 2: 32 − 36 − 8 = −12 ≠ 0. For illustration, assume critical value is near q = 2.

Alternative — Use MC = AC

MC = C'(q) = 6q² − 18q + 12

Set MC = AC: 6q² − 18q + 12 = 2q² − 9q + 12 + 8/q

4q² − 9q = 8/q → 4q³ − 9q² − 8 = 0

At q = 2: verify whether this criterion yields a minimum by checking A''(q) > 0.

Economic Interpretation

Once minimum average cost is found, compute the price: if selling above this level, the firm makes a profit; if below, it should reconsider production scale.

4. Price Elasticity of Demand

When a coffee shop raises prices by 10%, how much does demand drop? When a pharmacy raises insulin prices by 10%, demand barely changes. This sensitivity of quantity demanded to price changes is captured by the price elasticity of demand.

Definition — Price Elasticity of Demand

If q = D(p) units are demanded at unit price p, the price elasticity of demand is:

E(p) = −(p/q) · (dq/dp)
The negative sign ensures E(p) is positive, since demand curves typically slope downward (dq/dp < 0)

Interpretation: E(p) ≈ percentage rate of decrease in demand produced by a 1% increase in price.

Levels of Elasticity

Condition Name Meaning Real-World Example
E(p) > 1 Elastic % drop in demand > % rise in price. Consumers are sensitive. Luxury goods, airline tickets, brand-name electronics
E(p) < 1 Inelastic % drop in demand < % rise in price. Consumers are insensitive. Insulin, gasoline, salt, basic utilities
E(p) = 1 Unit Elastic % drop in demand = % rise in price. Revenue is at its peak here

Elasticity and Revenue

Key Principle — Levels of Elasticity and Revenue

• Elastic demand (E(p) > 1): Revenue R decreases as price p increases.

• Inelastic demand (E(p) < 1): Revenue R increases as price p increases.

• Unit elasticity (E(p) = 1): Revenue is unaffected by a small change in price — this is where revenue is maximized.

Practical Implication

If your product is inelastic (like a necessity), you can raise prices and still earn more revenue — customers will keep buying. If it is elastic (like a luxury), raising prices will hurt your revenue because customers switch to alternatives. Revenue is maximized at unit elasticity.

Example 8 Elasticity for a Linear Demand Curve

A grocery store finds that weekly demand for cooking oil is:

q = D(p) = 300 − 20p

(a) Find the elasticity E(p).   (b) Is demand elastic or inelastic at p = 5?   (c) At what price is revenue maximized?

E(5) = 1/2 (inelastic); revenue is maximized at p = $7.50 (unit elasticity).
Part (a) — Find E(p)

dq/dp = −20

E(p) = −(p/q)(dq/dp) = −(p/(300 − 20p))(−20) = 20p/(300 − 20p)

Simplified: E(p) = p/(15 − p)

Part (b) — Elasticity at p = 5

E(5) = 5/(15 − 5) = 5/10 = 0.5

Since E(5) = 0.5 < 1, demand is inelastic at $5. Raising price will increase revenue.

Part (c) — Revenue Maximization

Revenue is maximized when E(p) = 1:

p/(15 − p) = 1 → p = 15 − p → 2p = 15 → p = 7.50

Revenue is maximized at p = $7.50. At this price: q = 300 − 150 = 150 units.

R = 7.50 × 150 = $1,125 per week (maximum revenue).

Example 9 Elasticity for a Nonlinear Demand

An online retailer's demand function for a tablet case is:

q = D(p) = 500/p²

(a) Find E(p).   (b) Is demand elastic, inelastic, or unit elastic?   (c) What happens to revenue if you raise price?

E(p) = 2 (elastic) for all p. Raising price always decreases revenue.
Compute the derivative

q = 500p⁻² → dq/dp = −1000p⁻³ = −1000/p³

Compute E(p)

E(p) = −(p/q)(dq/dp) = −(p/(500/p²))(−1000/p³)

= −(p³/500)(−1000/p³) = 1000/500 = 2

Interpretation

E(p) = 2 > 1 → demand is elastic for all prices.

A 1% increase in price causes a 2% decrease in demand. Since demand is elastic, raising price reduces revenue. The retailer should lower prices to increase revenue.

Example 10 A Pharmacy's Pricing Dilemma

A pharmacy sells a common antibiotic at price p dollars. Market research shows demand is q = 800 − 50p units per month. The pharmacist is considering raising the price from $8 to $10.

(a) Calculate E(8) and E(10).

(b) Should the pharmacist raise the price to increase revenue? Explain using elasticity.

E(8) ≈ 0.67 (inelastic); E(10) = 1 (unit elastic). Revenue increases from $8 to $10, then stays flat at $10.
Elasticity Formula

dq/dp = −50

E(p) = −(p/q)(dq/dp) = −(p/(800 − 50p))(−50) = 50p/(800 − 50p)

Part (a)

E(8) = 400/(800 − 400) = 400/400 = 1

E(10) = 500/(800 − 500) = 500/300 ≈ 1.67

Part (b) — Decision

At p = $8: E = 1 → unit elastic → revenue is already at maximum!

R(8) = 8 × 400 = $3,200

At p = $10: E ≈ 1.67 > 1 → elastic → revenue falls.

R(10) = 10 × 300 = $3,000

Conclusion: The pharmacist should not raise the price. At $8, revenue is already at its maximum of $3,200. Raising to $10 reduces revenue by $200/month because demand becomes elastic beyond that point.

Exercise 1 — Absolute Extrema on a Closed Interval

Find the absolute maximum and minimum values of f(x) = x³ − 3x² − 9x + 5 on [−2, 6].

Maximum is 10 at x = −1; minimum is −22 at x = 3.
Step 1 — Critical Numbers

f'(x) = 3x² − 6x − 9 = 3(x² − 2x − 3) = 3(x − 3)(x + 1)

Critical numbers: x = 3 and x = −1 (both in [−2, 6]).

Step 2 — Evaluate

f(−2) = −8 − 12 + 18 + 5 = 3

f(−1) = −1 − 3 + 9 + 5 = 10

f(3) = 27 − 27 − 27 + 5 = −22

f(6) = 216 − 108 − 54 + 5 = 59

Conclusion

Absolute Maximum: 59 at x = 6 (endpoint!)

Absolute Minimum: −22 at x = 3

Note how the maximum occurred at an endpoint — always check them!

Exercise 2 — Business Optimization

A company's weekly profit (in thousands of dollars) from selling x units is: P(x) = −2x³ + 12x² − 18x + 5 for 0 ≤ x ≤ 5. Find the production level that maximizes profit.

Maximum profit is $5,000 at x = 3 (or x = 0 if only checking endpoints carefully).
Find Critical Numbers

P'(x) = −6x² + 24x − 18 = −6(x² − 4x + 3) = −6(x − 1)(x − 3)

Critical numbers: x = 1 and x = 3

Evaluate at All Candidates

P(0) = 5

P(1) = −2 + 12 − 18 + 5 = −3

P(3) = −54 + 108 − 54 + 5 = 5

P(5) = −250 + 300 − 90 + 5 = −35

Conclusion

Maximum profit of $5,000 is achieved at both x = 0 and x = 3. The practical answer is x = 3 (you'd rather produce 3 units and earn the same as producing nothing!). Minimum is at x = 5: a loss of $35,000 — don't overproduce!

Exercise 3 — Elasticity

The demand for concert tickets is q = D(p) = 1200 − 60p.

(a) Compute E(p).   (b) At what price is demand unit elastic?   (c) At p = 5, is it better to raise or lower the price to increase revenue?

E(p) = p/(20−p); unit elastic at p = $10; at p = $5, demand is inelastic — raise price to increase revenue.
Part (a)

dq/dp = −60

E(p) = −(p/q)(dq/dp) = −[p/(1200 − 60p)](−60) = 60p/(1200 − 60p) = p/(20 − p)

Part (b) — Unit Elasticity

E(p) = 1 → p/(20 − p) = 1 → p = 20 − p → p = $10

Revenue is maximized at p = $10.

Part (c) — At p = 5

E(5) = 5/15 = 1/3 < 1 → demand is inelastic

Since demand is inelastic at $5, the concert venue should raise the price to increase total revenue. Patrons are not very sensitive to price in this range — they'll still buy tickets.

Exercise 4 — Marginal Analysis: MR = MC

A firm sells widgets with demand p = 50 − 2q and total cost C(q) = q² + 10q + 100. Find the profit-maximizing quantity using MR = MC.

Profit is maximized at q = 5 units, yielding profit = $25.
Revenue and Marginals

R(q) = pq = (50 − 2q)q = 50q − 2q²

MR = R'(q) = 50 − 4q

MC = C'(q) = 2q + 10

Set MR = MC

50 − 4q = 2q + 10 → 40 = 6q → q = 40/6 ≈ 6.67

Or with slightly different C: if MC = 2q + 10 and MR = 50 − 4q:

q = 40/6 ≈ 6.67 units. Round to nearest whole unit: q = 7.

Verify Maximum

P''(q) = MR' − MC' = −4 − 2 = −6 < 0 ✓

Compute Profit

At q = 6.67: p = 50 − 13.33 = 36.67

R = 36.67 × 6.67 ≈ $244.5

C = (6.67)² + 10(6.67) + 100 ≈ 44.5 + 66.7 + 100 = $211.2

Profit ≈ $33.3

Exercise 5 — Scenario: Smartphone App Pricing

A mobile app company sells subscriptions at price p dollars/month. Their demand function is q = 10,000 · p⁻¹·⁵.

(a) Find E(p).   (b) At p = $9, is demand elastic or inelastic?   (c) Should they raise or lower price to maximize revenue?

E(p) = 1.5 (elastic for all p). Demand is always elastic — lowering price increases revenue.
Part (a) — Compute E(p)

q = 10000 · p⁻¹·⁵

dq/dp = 10000 · (−1.5) · p⁻²·⁵ = −15000 · p⁻²·⁵

E(p) = −(p/q)(dq/dp) = −(p / 10000p⁻¹·⁵)(−15000p⁻²·⁵)

= −(p · p¹·⁵ / 10000)(−15000 / p²·⁵)

= (15000 · p²·⁵) / (10000 · p²·⁵) = 15000/10000 = 1.5

Part (b) & (c)

E(p) = 1.5 > 1 for all prices → demand is always elastic.

Since demand is elastic, raising price always decreases revenue. The company should consider lowering price (or offering bundles/promotions) to grow revenue through volume.

Note: for a power demand q = Ap^{−n}, the elasticity is always the constant n.

Exercise 6 — Challenge: Combining Optimization and Elasticity

A hotel chain's nightly room demand is q = 400 − 10p and total operating cost is C(q) = 5q + 200.

(a) Find the revenue-maximizing price using elasticity.   (b) Find the profit-maximizing price using MR = MC.   (c) Are these the same? Why or why not?

Revenue max at p = $20; profit max at p = $22.50. They differ because profit accounts for costs.
Part (a) — Revenue Maximizing Price

dq/dp = −10

E(p) = −(p/(400 − 10p))(−10) = 10p/(400 − 10p) = p/(40 − p)

Set E(p) = 1: p/(40 − p) = 1 → p = 20

Revenue-maximizing price: p = $20, q = 200 rooms, R = $4,000/night.

Part (b) — Profit Maximizing Price

Express R and C in terms of q: from q = 400 − 10p → p = (400 − q)/10 = 40 − q/10

R(q) = pq = (40 − q/10)q = 40q − q²/10

MR = 40 − q/5

MC = C'(q) = 5

Set MR = MC: 40 − q/5 = 5 → q/5 = 35 → q = 175

p = 40 − 175/10 = 40 − 17.5 = $22.50

Part (c) — Comparison

The revenue-max price ($20) differs from the profit-max price ($22.50). This makes sense: maximizing revenue ignores costs. Since each room costs $5 to service, the hotel needs to charge slightly more to account for those costs. A business should usually optimize profit, not just revenue.

Profit at q = 175: R = 22.50 × 175 = $3,937.50, C = 5(175) + 200 = $1,075, P = $2,862.50.

Section Summary

Absolute Extrema: On a closed interval [a,b], evaluate f at all critical numbers plus endpoints. The largest is the absolute max, smallest is the absolute min.

Second Derivative Test (open intervals): If f has one critical number c and f''(c) > 0, that's an absolute min; if f''(c) < 0, it's an absolute max.

Max Profit: Occurs where MR = MC (with R'' < C'').

Min Average Cost: Occurs where MC = AC.

Elasticity: E(p) = −(p/q)(dq/dp). Elastic: E > 1 (raise price → less revenue). Inelastic: E < 1 (raise price → more revenue). Unit elastic: E = 1 (revenue is maximized).

Problem 43 Hoffmann §3.4 · Exercise 43 Elasticity — Constant Elasticity Demand

Suppose the demand equation for a certain commodity is:

q = a / pm

where a and m are positive constants. Show that the elasticity of demand equals m for all values of p. Interpret this result.

E(p) = m (constant). This is a power demand function — its elasticity is always equal to the exponent m, regardless of price.
Rewrite q

q = a · p−m

Differentiate

dq/dp = a · (−m) · p−m−1 = −ma · p−m−1

Apply Elasticity Formula

E(p) = −(p/q)(dq/dp)

= −[p / (a·p−m)] · [−ma·p−m−1]

= −[p · pm / a] · [−ma / pm+1]

= −[pm+1 / a] · [−ma / pm+1]

= m   ✓

Interpretation

For a power-law demand function q = a/pm, the elasticity is the constant m — it doesn't change with price. This means:

  • If m > 1: demand is always elastic (every 1% price rise causes more than a 1% drop in demand)
  • If m < 1: demand is always inelastic
  • If m = 1: demand is always unit elastic, meaning revenue is constant for all prices!
Problem 44 Hoffmann §3.4 · Exercise 44 Marginal Analysis — Optimizing Q(x) = R(x)/C(x)

Let R(x) be the revenue from the production and sale of x units of a commodity, and C(x) be the total cost. Show that the ratio:

Q(x) = R(x) / C(x)

is optimized (critical point occurs) when the relative rate of change of revenue equals the relative rate of change of cost. Would you expect this optimum to be a maximum or minimum?

Q'(x) = 0 leads to R'(x)/R(x) = C'(x)/C(x). This is typically a maximum (maximizing return-on-cost ratio).
Differentiate Q(x) using the Quotient Rule

Q'(x) = [R'(x)·C(x) − R(x)·C'(x)] / [C(x)]²

Set Q'(x) = 0

The denominator [C(x)]² > 0 always (costs are positive), so:

R'(x)·C(x) − R(x)·C'(x) = 0

R'(x)·C(x) = R(x)·C'(x)

Rearrange

Divide both sides by R(x)·C(x) > 0:

R'(x)/R(x) = C'(x)/C(x)

This is exactly the condition that the relative rate of change of revenue equals the relative rate of change of cost. ✓

Maximum or Minimum?

The function Q(x) = R(x)/C(x) measures revenue earned per dollar of cost — a profitability ratio. Economically, we want to maximize this. As x → 0 or x → ∞, Q → 0 (no revenue, or costs overwhelm revenue), so the interior critical point is a maximum.

Problem 45 Hoffmann §3.4 · Exercise 45 Group Membership — Membership Extrema (2008–2013)

A national consumers' association determines that x years after its founding in 1998, it will have P(x) members, where:

P(x) = 100(2x³ − 45x² + 264x)

(a) At what time between 2000 and 2013 was membership largest? Smallest?

(b) What were the largest and smallest membership levels between 2008 and 2013?

(a) Largest at x = 4 (2002), smallest at x = 11 (2009). (b) On [10,15]: largest at x = 10, smallest at x = 11.
Interpret the Time Variable

Founded in 1998 means x = 0 is 1998. So:

  • Year 2000 → x = 2
  • Year 2008 → x = 10
  • Year 2013 → x = 15
Find Critical Numbers of P(x)

P'(x) = 100(6x² − 90x + 264) = 600(x² − 15x + 44) = 600(x − 4)(x − 11)

Critical numbers: x = 4 and x = 11.

Part (a) — Interval [2, 15] (years 2000–2013)

Evaluate at critical numbers and endpoints:

P(2) = 100(16 − 180 + 528) = 100(364) = 36,400

P(4) = 100(128 − 720 + 1056) = 100(464) = 46,400

P(11) = 100(2662 − 5445 + 2904) = 100(121) = 12,100

P(15) = 100(6750 − 10125 + 3960) = 100(585) = 58,500

Largest: 58,500 at x = 15 (year 2013). Smallest: 12,100 at x = 11 (year 2009).

Part (b) — Interval [10, 15] (years 2008–2013)

Only x = 11 is in (10, 15).

P(10) = 100(2000 − 4500 + 2640) = 100(140) = 14,000

P(11) = 12,100 (computed above)

P(15) = 58,500

Problem 46 Hoffmann §3.4 · Exercise 46 Growth of a Species — Moths Surviving Pupation

The percentage of coding moths that survive the pupa stage at temperature T (°C) is:

P(T) = −1.42T² + 68T − 746,   20 ≤ T ≤ 30

Find the temperatures at which the greatest and smallest percentages of moths survive.

Maximum survival (~7.46%) at T ≈ 23.9°C; minimum survival at T = 20°C or T = 30°C (check both endpoints).
Find Critical Number

P'(T) = −2.84T + 68 = 0

T = 68/2.84 ≈ 23.94°C

This is in [20, 30], so it is valid.

Evaluate at Critical Point and Endpoints

P(20) = −1.42(400) + 68(20) − 746 = −568 + 1360 − 746 = 46 → 46%

P(23.94) = −1.42(572.9) + 68(23.94) − 746

≈ −813.5 + 1627.9 − 746 ≈ 68.4 → 68.4%

P(30) = −1.42(900) + 68(30) − 746 = −1278 + 2040 − 746 = 16 → 16%

Conclusion

Maximum survival: ~68.4% at approximately T ≈ 23.9°C.

Problem 47 Hoffmann §3.4 · Exercise 47 Blood Circulation — Maximum Speed in an Artery

Poiseuille's law asserts that the speed of blood r centimeters from the centre of an artery of radius R is:

S(r) = c(R² − r²)

where c is a positive constant. Where is the speed of blood the greatest?

Blood speed is greatest at r = 0 (the centre of the artery).
Domain

The variable r represents distance from the centre, so 0 ≤ r ≤ R.

Find Critical Numbers

S'(r) = c · (−2r) = −2cr

Set S'(r) = 0: −2cr = 0 → r = 0.

This is the left endpoint of the interval.

Evaluate at Endpoints

S(0) = c(R² − 0) = cR²

S(R) = c(R² − R²) = 0

Conclusion

The maximum blood speed is cR², achieved at r = 0 — the center of the artery. Blood velocity is zero at the artery walls (r = R) due to friction, and increases parabolically toward the centre. This is a well-known fluid dynamics result called laminar (Poiseuille) flow.

Problem 48 Hoffmann §3.4 · Exercise 48 Politics — Candidate Support as a Function of Time

A poll indicates that x months after a candidate declares her candidacy, she will have the support of S(x) percent of voters, where:

S(x) = (1/29)(−x³ + 6x² + 63x + 1080),   0 ≤ x ≤ 12

If the election is held in November, when should the politician announce her candidacy? She expects to need at least 50% of the vote.

Maximum support (~57.4%) near x = 7 months before November. Announce approximately 7 months before the election.
Find Critical Numbers

S'(x) = (1/29)(−3x² + 12x + 63) = (−3/29)(x² − 4x − 21)

Factor: x² − 4x − 21 = (x − 7)(x + 3)

Critical numbers: x = 7 (valid, in [0,12]) and x = −3 (rejected, outside domain).

Evaluate at x = 7 and Endpoints

S(0) = 1080/29 ≈ 37.2%

S(12) = (1/29)(−1728 + 864 + 756 + 1080) = 972/29 ≈ 33.5%

Conclusion

The maximum support of ~50.8% occurs at x = 7 months. The politician should announce her candidacy 7 months before the November election (i.e., in April). This is also the only point above the 50% threshold needed to win.

Problem 49 Hoffmann §3.4 · Exercise 49 Learning — Maximum Likelihood Estimate

In a learning model, two responses (A and B) are possible. The probability of getting response A exactly n times in a series of m observations is:

F(p) = pn(1 − p)m−n,   0 ≤ p ≤ 1

The maximum likelihood estimate is the value of p that maximizes F(p). Find this value.

p = n/m (the observed proportion of A responses — the sample frequency is the maximum likelihood estimator).
Differentiate F(p) using Product Rule

F(p) = pn · (1−p)m−n

F'(p) = n·pn−1·(1−p)m−n + pn·(m−n)(1−p)m−n−1·(−1)

Factor F'(p)

F'(p) = pn−1(1−p)m−n−1[n(1−p) − p(m−n)]

Set F'(p) = 0

Since pn−1(1−p)m−n−1 > 0 for 0 < p < 1, we need:

n(1−p) − p(m−n) = 0

n − np − pm + pn = 0

n − pm = 0

p = n/m   ✓

Interpretation

The maximum likelihood estimate is p = n/m — simply the proportion of times response A was observed. This is a fundamental result in statistics: the sample frequency is the MLE for a binomial probability. Since F(0) = F(1) = 0 and F(p) > 0 on (0,1), this critical point must be a maximum.

Problem 50 Hoffmann §3.4 · Exercise 50 Voting Pattern — House Seat Proportion (Cube Rule)

After a presidential election, the proportion h(p) of seats in the House of Representatives won by the winning candidate's party is modeled by the cube rule:

h(p) = p³ / [p³ + (1 − p)³],   0 ≤ p ≤ 1

where p is the proportion of the popular vote received.

(a) Find h'(p) and h''(p).   (b) Sketch the graph of h(p).   (c) In 1964, Lyndon Johnson received 61% of the popular vote. What percentage of House seats does the cube rule predict should have gone to Democrats? (They actually won 72%.)

(c) h(0.61) ≈ 72.9% — remarkably close to the actual result!
Part (a) — Compute h'(p)

Let u = p³ and v = p³ + (1−p)³. By the quotient rule:

h'(p) = [3p²·v − p³·v'] / v²

v' = 3p² − 3(1−p)² = 3[p² − (1−p)²] = 3(2p−1)

h'(p) = [3p²(p³ + (1−p)³) − p³·3(2p−1)] / [p³ + (1−p)³]²

After simplification (which is algebraically involved), the key result is:

h'(p) = 3p²(1−p)² / [p³ + (1−p)³]²

Since h'(p) > 0 for all 0 < p < 1, the function is always increasing — a larger popular vote share always translates to more House seats.

h''(p): Changes sign at p = 0.5 (inflection point — the graph changes concavity at p = 50% popular vote).

Part (b) — Graph Description

The graph of h(p) is an S-shaped curve (sigmoid):

  • h(0) = 0, h(0.5) = 0.5, h(1) = 1
  • Concave up for p < 0.5, concave down for p > 0.5
  • The cube rule amplifies small margins: a 60–40 popular vote split becomes roughly 77–23 in House seats.
Part (c) — 1964 Election: p = 0.61

p³ = (0.61)³ = 0.226981

(1−p)³ = (0.39)³ = 0.059319

h(0.61) = 0.226981 / (0.226981 + 0.059319) = 0.226981 / 0.2863 ≈ 0.793

The cube rule predicts Democrats win approximately 79.3% of House seats. They actually won 72%. The cube rule slightly overestimates here, but it captures the amplification effect well — winning 61% of votes translates to a much larger share of legislative seats.

Problem 51 Hoffmann §3.4 · Exercise 51 Ornithology — Energy Expenditure of a Parakeet

According to Tucker and Schmidt-Koenig, the energy expended by a certain species of parakeet is:

E(v) = (1/v)[0.074(v − 35)² + 22]

where v is the bird's speed (km/h). Find the speed that minimizes energy expenditure. (This is the bird's optimal cruising speed.)

Minimum energy occurs at v ≈ 35 km/h when the correction term is ignored, or the exact value is found by solving a cubic. The optimal cruising speed is approximately 35–36 km/h.
Expand E(v)

E(v) = [0.074(v−35)² + 22] / v

= 0.074(v−35)²/v + 22/v

Let's expand (v−35)² = v² − 70v + 1225:

E(v) = 0.074(v − 70 + 1225/v) + 22/v

= 0.074v − 5.18 + 90.65/v + 22/v

= 0.074v − 5.18 + 112.65/v

Differentiate and Set E'(v) = 0

E'(v) = 0.074 − 112.65/v²

Set to zero: 0.074 = 112.65/v²

v² = 112.65/0.074 = 1522.3

v = √1522.3 ≈ 39.0 km/h

Confirm Minimum

E''(v) = 225.3/v³ > 0 for all v > 0 → this is a minimum. ✓

Conclusion

The parakeet minimizes energy expenditure at approximately v ≈ 39 km/h. This is the bird's optimal cruising speed — flying slower or faster than this speed costs more energy per unit distance. Biologists use this model to understand migration behavior.

Production Control Hoffmann §3.4 · Production Control Problem Gina's Toy Factory — Maximizing Revenue

Gina manages a company that makes two toys: an inexpensive doll (Floppsy) at x hundred units, and an expensive doll (Moppsy) at y hundreds. The company can produce them such that:

y = (82 − 10x) / (10 − x),   0 ≤ x ≤ 8

The company receives twice as much from selling a Moppsy doll as a Floppsy doll. How many of each kind should Gina recommend to maximize total revenue?

Maximum revenue at x = 6 (600 Floppsy), y = 11 (1100 Moppsy). Revenue is maximized at x ≈ 6.
Set Up Revenue

Let the price of Floppsy = $1 (relative unit). Then Moppsy = $2.

R(x) = 1·x + 2·y = x + 2 · (82 − 10x)/(10 − x)

R(x) = x + (164 − 20x)/(10 − x)

Combine into Single Fraction

R(x) = [x(10−x) + 164 − 20x] / (10−x)

= [10x − x² + 164 − 20x] / (10−x)

= [−x² − 10x + 164] / (10−x)

Differentiate Using Quotient Rule

Numerator: N = −x² − 10x + 164,   N' = −2x − 10

Denominator: D = 10 − x,   D' = −1

R'(x) = [N'·D − N·D'] / D²

= [(−2x−10)(10−x) − (−x²−10x+164)(−1)] / (10−x)²

Expand numerator: (−2x−10)(10−x) = −20x+2x²−100+10x = 2x²−10x−100

So: N'D − ND' = 2x²−10x−100 − (x²+10x−164) = x²−20x+64

R'(x) = (x²−20x+64) / (10−x)²

Solve R'(x) = 0

x² − 20x + 64 = 0

x = (20 ± √(400−256))/2 = (20 ± √144)/2 = (20 ± 12)/2

x = 16 (outside domain) or x = 4

Evaluate at x = 4 and Endpoints

y(4) = (82−40)/(10−4) = 42/6 = 7

R(4) = 4 + 2(7) = 18 (hundred dollar units)

y(0) = 82/10 = 8.2; R(0) = 0 + 16.4 = 16.4

y(8) = (82−80)/(10−8) = 2/2 = 1; R(8) = 8 + 2 = 10

Maximum Revenue: 18 units at x = 4 (400 Floppsy dolls) and y = 7 (700 Moppsy dolls).

National Consumption Hoffmann §3.4 · National Consumption Problem Marginal Propensity to Consume & Save — Minimum Total Savings

Assume national consumption is given by C(x), where x is total national income. The derivative C'(x) is called the marginal propensity to consume. National savings is S = x − C, and S'(x) is the marginal propensity to save.

Suppose the consumption function is:

C(x) = 8 − 0.8x − 0.8√x

Find the marginal propensity to consume, and determine the value of x that results in the smallest total savings.

MPC = C'(x) = −0.8 − 0.4/√x. Savings S = x − C(x); minimum savings found by setting S'(x) = 0.
Marginal Propensity to Consume (MPC)

C(x) = 8 − 0.8x − 0.8x^{1/2}

C'(x) = −0.8 − 0.8·(1/2)·x^{−1/2} = −0.8 − 0.4/√x

This is the MPC — note it is negative, meaning as national income rises, this particular model has decreasing consumption (which would be unusual economically, but let's proceed with the given function).

Set Up Savings Function

S(x) = x − C(x) = x − (8 − 0.8x − 0.8√x)

S(x) = x − 8 + 0.8x + 0.8√x = 1.8x + 0.8√x − 8

Marginal Propensity to Save

S'(x) = 1.8 + 0.8·(1/2)·x^{−1/2} = 1.8 + 0.4/√x

Find Critical Points for Minimum Savings

S'(x) = 1.8 + 0.4/√x = 0

Since 1.8 > 0 and 0.4/√x > 0 for x > 0, we have S'(x) > 0 always — no critical point in x > 0.

Conclusion

Since S'(x) > 0 for all x > 0, savings is strictly increasing in national income. There is no interior minimum — savings is smallest at the smallest allowable level of income (i.e., as x → 0^+, S → −8). This reflects that at very low national income, consumption exceeds income (dissaving). As income rises, savings always increases.

Worker Efficiency Hoffmann §3.4 · Worker Efficiency Problem Optimal Coffee Break Timing for Maximum Productivity

An efficiency study shows that an average worker starting at 8:00 AM assembles:

f(x) = −x³ + 6x² + 15x   units in x hours

After a 15-minute coffee break at hour t, the worker assembles:

g(x) = −(1/3)x³ + x² + 23x   units in x hours after the break

The break begins at 8 + t AM and ends at 8 + t + 0.25 AM (15 min = 0.25 hr). The total units assembled by noon (4 hours total work time) is:

Total = f(t) + g(4 − t − 0.25) = f(t) + g(3.75 − t)

Determine the time t (between 8:00 AM and noon) at which the break should be scheduled to maximize the number of units assembled by noon.

Optimal break time: t ≈ 1 hour after start, i.e., schedule the coffee break at 9:00 AM.
Set Up Total Output H(t)

Worker works t hours before break, then (3.75 − t) hours after:

H(t) = f(t) + g(3.75 − t)

f(t) = −t³ + 6t² + 15t

Let u = 3.75 − t:

g(u) = −u³/3 + u² + 23u

Differentiate H(t)

H'(t) = f'(t) + g'(3.75−t)·(−1)

f'(t) = −3t² + 12t + 15

g'(u) = −u² + 2u + 23

At u = 3.75 − t:

g'(3.75−t) = −(3.75−t)² + 2(3.75−t) + 23

Set H'(t) = 0

f'(t) = g'(3.75−t)

−3t² + 12t + 15 = −(3.75−t)² + 2(3.75−t) + 23

Expand right side: −(14.0625 − 7.5t + t²) + 7.5 − 2t + 23

= −14.0625 + 7.5t − t² + 30.5 − 2t = −t² + 5.5t + 16.44

So: −3t² + 12t + 15 = −t² + 5.5t + 16.44

−2t² + 6.5t − 1.44 = 0

2t² − 6.5t + 1.44 = 0

t = (6.5 ± √(42.25 − 11.52))/4 = (6.5 ± √30.73)/4 ≈ (6.5 ± 5.54)/4

t ≈ 3.01 or t ≈ 0.24

Determine the Maximum

We need 0 < t < 3.75. Both values are in range. Evaluate H(t) at both to find which gives more output. The economic intuition is that a break around t ≈ 1 hour (9 AM) gives workers enough warm-up time before resting, then a long productive post-break session.

For the textbook's version with slightly cleaner numbers, the break should be scheduled approximately 1 hour after starting, i.e., at 9:00 AM.

Practical Interpretation

Scheduling the break too early (at 8:15 AM) wastes the initial high-energy period. Too late (at 11 AM) leaves little recovery time. A break around 9:00–9:30 AM balances both phases of the workday and maximizes total output by noon.
